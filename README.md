# Claude Chat Knowledge Base

A personal knowledge management system for organizing and reusing Claude Code CLI responses.

## Purpose

Store and organize useful Claude Code CLI interactions as reference documentation for future projects. Each topic folder contains chat transcripts saved as markdown files.

## Structure

```
topic-name/
└── YYYY-MM-DD-description.md
```

- **Topic folders** - Organize by project, technology, or subject
- **Date-based filenames** - Easy chronological tracking
- **Markdown format** - Readable and version-controllable

## Quick Start

1. Clone this repository
2. Add `AGENT.md` with your custom Claude guidelines (optional)
3. Start saving your Claude Code CLI responses to topic folders
4. Use as reference for future work
